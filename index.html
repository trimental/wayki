A non-offical wayland wikipedia

Working with display server protocols is hard work, however a large part of that work is documentation.

Wayki tries to document the wayland protocol and provide code examples in Rust using the Smithay project.

Visit Wiki

Offical documentation also exists...

The wayland protocol of course has offical documentation however Wayki is designed to be more user readable and trys to provide useful code examples.

There's also the wayland-rs developer docs, meant for coding but can also provide insight into the protocols workings.
